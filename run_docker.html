Run With Docker

To simply test with mpi, we can simulate the parallel environment by running Docker with multiple containers.

Just run:

cd psvm/docker
./run_docker.sh 3   # build docker containers, Usage: ./run_docker.sh NODE_NUM
./run_psvm.sh 3 splice 0.1    # run psvm parallelly, USAGE: ./run_psvm NODE_NUM DATA_NAME RANK_RATIO
./clean_docker.sh 3   # clean docker containers, Usage: ./clean_docker.sh NODE_NUM
  • NODE_NUM: the number of mpi nodes, including master.
  • DATA_NAME: filename of the training data, filename of testing data should be DATA_NAME.t. Both training and testing data should be under the directory psvm/data.
  • RANK_RATIO: parameter for psvm, please refer to command-line flags.
  • After run psvm, models are stored under psvm/models, and result file is under psvm directory.
